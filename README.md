neotree-lbw-svn
NeoTree Low Birth Weight (LBW) Small Vulnerable Newborns (SVN) analysis repository from UCL GOS Institute of Child Health. This project focuses on data analysis and modeling for neonatal outcomes in low-resource settings, particularly for LBW infants (1.5-2.5kg).

Project Overview
NeoTree is an open-source digital platform for newborn care, providing clinical decision support, data capture, and quality improvement tools in low-resource environments like Zimbabwe and Malawi. This repo analyzes LBW/SVN cohorts in the hospitals Sally Mugabe Children's Hospital in Zimbabwe (SMCH) and Kamuzu Central Hospital (KCH) in Malawi.
